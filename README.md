# Book-Store

A Bookstore project for managing books, authors, and sales, allowing users to browse, search, and purchase books online, while admins can manage inventory and orders efficiently.

## Installation
Steps to install your project:
```bash
# Clone the repo
git clone https://github.com/username/projectname.git

# Go to project folder
cd projectname

# Install dependencies
npm install

# add .env file to backend

MONGO_URI=PASTE_MONGO_DB_URL_HERE
JWT_SECRET=YOUR_SECRET_KEY
PORT=5000

## to run frontend

cd frontend-bookstore

npm run dev


# open new terminal 
## to run backend

cd backend-bookstore

npm run start
